Connecting Disinformation with tidygraph

I recently participated in a hackathon organised by EU’s anti-disinformation task force where they gave us access to their data base. The data base consists of all disinformation cases the group has collected since it started in 2015. Their data can also be browsed online on their web page www.euvsdisinfo.eu. I built a small R package that makes it easier to download the data from their API (it seems though that the API is currently not updated as regularly as their web page) if you want to check out the data yourself. »
